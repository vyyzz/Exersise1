# Linked List Library

This C++ library provides functions for working with linked lists, including building new linked lists, retrieving data item values, printing linked lists, updating data in linked lists, and deleting entire linked lists.

## Getting Started

### Prerequisites
Make sure you have the following installed on your system:

- C++ compiler (e.g., g++)

### Clone and Compile
Follow these steps to clone and compile the library:

```bash
git clone https://github.com/vyyzz/Linked_List_Library.git
cd Linked_List_Library
g++ -o linked_list_app linked_list.cpp unit_test_linked_list.cpp
./linked_list_app
